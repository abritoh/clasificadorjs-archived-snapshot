ClasificadorJS is an AngularJS-based application designed to assist law enforcement agencies in Mexico ─formerly the now-dissolved Federal Police─ with operational, classification and investigation tasks.

This front-end application utilizes a modular architecture, and served as a foundational-base for other projects. It enabled the easy implementation of consistent structures, by utilizing patterns, best angularjs coding-practices, templates, controllers, and components (directives) to be rehused across other developments.

Design logos, images, and business requirements are property of the final-application owner [IAPPP]. The author/publisher [ClusterBR] contributed to implement the general application architecture, AngularJS components, and application logic. For speed-up large-scale parallel development, front-end can be divided in the following roles: graphic designer, HTML-layout-designer (maquetador), front-end developer(s) and front-end developer lead.

Since this is a purely front-end application, it is disconnected from any databases or external data sources and therefore will not display, save and/or use any sensitive data. The information (news) shown was generated by using generative AI tools to populate the JSON files with global and Mexico-specific trending news.
